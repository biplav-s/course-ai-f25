** Illustration of BODMAS, also called PEMDAS rule **

BODMAS stands for Brackets, Orders, Division, Multiplication, Addition, and Subtraction, while PEMDAS is used in the U.S. and stands for Parentheses, Exponents, Multiplication, Division, Addition, and Subtraction. 


1. exp2.pddl   = b * c
2. exp3.pddl   = a + b * c
3. exp4.pddl   = a + b * c - d
4. exp5.pddl   = a + b * c - d / e
